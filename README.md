# Portfolio

Personal portfolio site — built with vanilla HTML/CSS/JS (no framework), deployed on Vercel.

**Live:** https://shivani-portfolio-two-pi.vercel.app

## Sections

- **About** — AI/ML Engineer & Python Developer background
- **Experience** — timeline of professional roles
- **Projects** — featured work with links to source
- **Skills** — interactive tech-stack breakdown by category
- **Education & Certifications**
- **Contact** — form (Formspree) plus direct links

## Stack

- Plain HTML5 / CSS3 / JavaScript — no build step, no dependencies
- Custom animations: typewriter effect, cursor glow, floating particles, animated stat counters
- Formspree for the contact form backend
- Deployed on Vercel

## Running locally

No build step required — it's static.

```bash
git clone https://github.com/shivani3291-lab/shivani-portfolio.git
cd shivani-portfolio
# open index.html directly, or serve it:
python -m http.server 8000
```
